# 📊 Exploratory Data Analysis (EDA) & Data Visualization 🚀

This repository contains my hands-on practice of **Exploratory Data Analysis (EDA)** and **Data Visualization** using Python.

It covers **univariate, bivariate, and multivariate analysis** along with multiple visualization techniques using **Matplotlib and Seaborn**.
---
# 📌 About This Repository
* Focused on **understanding data before modeling**
* Covers **real-world analysis workflow**
* Includes both **manual EDA and automated profiling**
* Designed to strengthen **data analysis skills for projects and interviews**

---

## 🔹 Types of Analysis

### 📌 Univariate Analysis

* Distribution of single variables
* Histograms, count plots

### 📌 Bivariate Analysis

* Relationship between two variables
* Scatter plots, bar charts

### 📌 Multivariate Analysis

* Analysis of multiple variables together
* Correlation, grouped visualizations
