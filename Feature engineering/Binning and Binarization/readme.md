---

## 🔹 Binning

* Converting continuous values into intervals/groups
* Reducing noise in data
* Improving interpretability

### Example:

```text id="7s5h5o"
Age → Young | Adult | Senior
```

---

## 🔹 Binarization

* Converting numerical values into binary values (0 or 1)
* Applying threshold-based transformation

### Example:

```text id="efn4mj"
if value > threshold:
    1
else:
    0
```
